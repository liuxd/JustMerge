# Just Merge

### Feature

+ Merge multiple repositories in the same time.
+ Real-time output.

### Dependence

[Node.js](http://nodejs.org/)

### Installation

```
git clone git@github.com:liuxd/JustMerge.git
cd JustMerge
npm install
```

### Let's start!

```
cd JustMerge
./boss.sh start
```
